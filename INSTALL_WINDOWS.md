# WinAutomate: Инструкция по установке (Windows)

Это приложение поможет вам управлять Windows с помощью голоса или текста через ИИ Gemini.

## 1. Требования
Для работы приложения на вашем компьютере должен быть установлен **Node.js** (версия 18 или выше).
- Скачать: [https://nodejs.org/](https://nodejs.org/)

## 2. Быстрый запуск
1. Скачайте проект (Export as ZIP) и распакуйте.
2. Запустите файл `run_local.bat` двойным кликом.
3. Скрипт **автоматически**:
   - Проверит и установит зависимости Node.js.
   - Установит библиотеки Python (Flask, CORS).
   - Запустит исполнитель команд (`agent_runner.py`) в **фоновом режиме**.
   - Откроет интерфейс в браузере.

## 4. Как обновлять проект через GitHub (рекомендуется)
Чтобы не скачивать ZIP-архив каждый раз:
1. В AI Studio нажмите на меню (три точки или иконка настроек) и выберите **"Export to GitHub"**.
2. В консоли на вашем компьютере клонируйте репозиторий:
   ```bash
   git clone <ваша_ссылка>
   ```
3. Для обновления просто пишите `git pull`.

## 5. Безопасность и Приватность
- **Только Localhost**: По умолчанию `run_local.bat` настроен так, что приложение доступно **только с вашего компьютера** (`127.0.0.1`). Даже если вы находитесь в публичной Wi-Fi сети, никто не сможет подключиться к вашему агенту извне.
- **Фоновый процесс**: Агент работает скрыто. Помните, что он имеет доступ к выполнению команд, поэтому никогда не передавайте свой API ключ или доступ к компьютеру третьим лицам.
